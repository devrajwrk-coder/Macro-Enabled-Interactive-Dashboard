# Macro-Enabled-Interactive-Dashboard
## Project Objective
This macro-enabled interactive Excel dashboard analyzes sales executive performance by city. Using macros across Dashboard 1 to 4, And dynamically displays top and bottom performers, target achievement %, and gap from target, allowing users to switch views effortlessly and quickly identify high performers and areas needing improvement.
